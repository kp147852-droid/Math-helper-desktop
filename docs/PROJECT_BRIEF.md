# Project Brief: Math Helper Desktop

## Problem
Students often need step-by-step support, targeted practice, and review loops in one place instead of fragmented tools.

## Solution
Math Helper Desktop combines solving, hinting, timed assessment, and missed-question review in a local desktop app.

## Business/education value
- Improves study feedback loops
- Supports differentiated practice by topic
- Provides local-first privacy and persistence

## Technical highlights
- SymPy-based explainable solving workflows
- Input normalization/classification pipeline
- SQLite persistence for review/test workflows
- Optional OCR and graphing integrations
