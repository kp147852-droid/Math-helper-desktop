# Resume Bullet Starters

- Built and shipped a Python desktop education app that supports algebra, precalculus, trigonometry, geometry, calculus, and linear algebra with step-by-step solutions and guided hinting.
- Designed an input normalization pipeline to convert noisy pasted homework text into parseable symbolic expressions, improving solver usability for real-world user inputs.
- Implemented topic-specific symbolic solve workflows with SymPy and fallback handling, enabling explainable output instead of answer-only responses.
- Developed SQLite-backed practice workflows including timed tests, missed-question review set generation, and export functionality for repeated learning loops.
- Added graph generation, OCR hooks, CI validation, and contributor templates to improve product completeness and engineering quality.
